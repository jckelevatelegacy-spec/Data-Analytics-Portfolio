# E-Commerce User Behavior & Retention Analysis

## Project Overview

This Excel project analyzes customer behavior across an e-commerce platform by evaluating conversion funnel performance and cohort-based retention trends.

The objective was to transform raw user activity logs into actionable business insights and understand how effectively the platform converts visitors into customers while retaining users over time.

## Business Questions

1. Where do users drop off within the purchasing funnel?

2. How effectively does the platform convert visitors into customers?

3. How does customer retention change over time?

4. Are newer customer cohorts performing differently than earlier cohorts?

## Tools Used

* Excel / Google Sheets
* Pivot Tables
* VLOOKUP
* Cohort Analysis
* Funnel Analysis

## Skills Demonstrated

* Funnel analysis
* Cohort analysis
* Retention analysis
* Data cleaning
* Spreadsheet modeling
* Business analysis

## Key Findings

* Significant user drop-off occurred between funnel stages of the purchasing journey.

* Customer retention declined over time, highlighting opportunities to improve long-term engagement.

* Cohort analysis revealed performance differences between acquisition periods.

* Funnel and retention metrics provided a clearer picture of customer behavior throughout the customer lifecycle.

## Recommendations

* Investigate stages with the highest user abandonment.

* Implement strategies to improve customer retention and repeat purchases.

* Continue monitoring cohort performance to identify long-term trends.

* Develop initiatives focused on increasing customer lifetime value.

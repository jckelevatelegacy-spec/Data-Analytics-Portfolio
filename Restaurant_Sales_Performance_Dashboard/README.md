# Restaurant Sales Performance Dashboard

## Project Overview

This Tableau dashboard analyzes restaurant sales performance to identify key revenue drivers, top-performing markets, and customer purchasing trends.

The project was designed to transform transactional restaurant data into actionable business insights through interactive visualizations and KPI analysis.

## Business Questions

1. Which cities generate the highest revenue?

2. Which restaurant categories drive the most sales?

3. How does customer behavior change over time?

4. Where are opportunities for business growth?

## Tools Used

* Tableau
* Excel / Google Sheets
* Data Visualization
* Business Analysis

## Skills Demonstrated

* Dashboard design
* KPI development
* Data storytelling
* Trend analysis
* Geographic analysis
* Business recommendations

## Key Findings

* Revenue generation varied considerably across markets.
* Sales performance was concentrated within specific categories.
* Customer purchasing trends changed over time.
* Certain markets presented stronger growth opportunities than others.

# Fresh Beats Marketing Analysis

## Project Overview

This Business Intelligence project analyzes user engagement, artist promotion effectiveness, and subscription conversion behavior for Fresh Beats, a music streaming platform focused on promoting emerging artists.

The objective was to identify opportunities to improve user engagement, optimize promotional strategies, and increase conversion from free to paid subscriptions.

## Business Questions

1. Which promotional strategies are most effective at increasing artist visibility?

2. What user behaviors are associated with higher engagement?

3. Which factors contribute to free-to-paid subscription conversions?

4. How can genre preferences improve user recommendations?

## Tools Used

* Google Sheets / Excel
* Data Visualization
* Business Analysis

## Skills Demonstrated

* Stakeholder reporting
* Data storytelling
* Insight interpretation
* Business recommendations
* User engagement analysis
* Marketing analysis

## Key Findings

* Certain promotional strategies generated stronger artist engagement than others.

* User behavior patterns revealed opportunities to improve subscription conversion rates.

* Genre preferences can be leveraged to improve recommendations and content discovery.

* Aligning promotional efforts with user preferences can improve both engagement and revenue generation.

## Recommendations

* Prioritize high-performing promotional campaigns.

* Develop targeted strategies to convert free users into paid subscribers.

* Personalize recommendations based on user listening preferences.

* Continue monitoring user engagement metrics over time.
